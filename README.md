# NUMPY_PROJECTS

This repository is a part of my AI/ML learning journey.

Instead of only watching tutorials, I wanted to learn **NumPy by building projects** and documenting everything I learn along the way. Every project in this repository is built to strengthen my understanding of NumPy and the concepts that form the foundation of Machine Learning and Deep Learning.

The goal of this repository is simple:

* Learn NumPy through hands-on projects.
* Practice writing clean and efficient code.
* Understand the "why" behind NumPy functions instead of memorizing them.
* Build a personal reference that I can revisit in the future.
* Track my progress from beginner-level projects to AI/ML implementations.
